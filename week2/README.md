# Video Resolution
> width x height of the frame
* SD is 640x360 or 720x480
* HD is 1280x720 ( also known as 720p)
* Full HD is 1920x1080 ( also known as 1080p)
* UHD is 3840x2160 ( also known as 4K)

# Frame Rate / FPS
* ex. 40 FPS: 40 images displayed per second

# Video Formats
* AVI, MP4
